# HR Management System (HRMS)

A full-stack Human Resource Management System built with React.js frontend and Node.js backend.

## Project Structure

- `frontend/` - React.js application
- `backend/` - Node.js/Express API server

## Quick Start

### Frontend

cd frontend
npm install
npm start

### Backend

cd backend
npm install
npm run dev



## GitHub Push Instructions

### 1. Initial Setup 

# Navigate to your project root
cd D:\react-projects\hrms

# Initialize git repository
git init

# Add all files to staging
git add .

# Make initial commit
git commit -m "Initial commit: HRMS application with React frontend and Node.js backend"

# Add your GitHub repository as remote origin
git remote add origin https://github.com/ShabnaUmmer/HRMS.git

# Push to GitHub
git push -u origin main
